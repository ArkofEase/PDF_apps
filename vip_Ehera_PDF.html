User Guide

About Ehera PDF

  • Personal Project: This application was created by an individual. Unauthorized distribution is prohibited.
  • Security: This app runs entirely in your browser (client-side). Your files are processed on your own computer and are never uploaded to a server, so you don't need to worry about data security.
  • Origin: Designed by a CPA office practitioner based on practical experience to address inconveniences found in other software.
  • Download Source: You can download the source code from my blog: arkofease.blogspot.com

Limitations & Adobe Comparison

While this tool offers unique features, it cannot replicate Adobe's proprietary technology. It is designed as a substitute, not a full replacement.

  • Password Protection: Documents protected by passwords cannot be opened due to Adobe's encryption standards.
  • Fillable Forms: Advanced features like Adobe Fillable Forms are not recognized by this app.
  • "Aditor" (Add + Editor): You can add text to forms, and text created within this app is editable. However, you cannot edit existing text in the PDF. To correct errors, use the White-out feature to cover the text or type the text using opacity (Auto-Filler App). (For copying text, free Adobe Reader is recommended).
  • Excel Export: Direct export to Excel is not supported. However, you can crop a table to a blank page and use Excel's "Data from PDF" feature (or AI tools) to import the data.

Tool Manuals

1. File Preparer

Merge Tab:
  • Combine multiple PDFs or Images into a single PDF.
  • Sorting: Drag & drop to reorder, or use "Sort" to auto-arrange based on keyword priority (e.g., W2, 1099).
  • Options: Generate a "Table of Contents" with clickable links and add page numbers to the footer of every page.
Extract Tab:
  • PDF: Save specific pages (e.g., "1-3, 5") from a document. Choose "Merge Result" to create a single file or "Split Result" for separate files.
Split Tab:
  • Divide a single page into multiple horizontal or vertical segments (e.g., separating receipts).
  • Select pages from the grid, click "Apply Split Tool", and adjust cut lines in the preview window.

2. PAGE ORGANIZER

Organizer (Page Management):
  • Reorder: Drag and drop pages to rearrange them.
  • Manage: Rotate, Delete, Duplicate pages, or Insert Blank pages using the toolbar.
  • Import: Combine multiple PDFs and images into one workspace.
Write Note:
  • You can leave a note at any desired location.

3. Redact OCR

Auto-Detection:
  • Keywords: Enter words like "Confidential" or specific numbers to find and mask them automatically.
  • SSN: Enable "Auto-detect SSN" to find Social Security Numbers. Choose between Full Mask (XXX-XX-XXXX) or Partial Mask (XXX-XX-****).
Manual Tools:
  • Drag to Redact: Simply click and drag over any text or area to mask it immediately.
  • Review: After scanning, a list of matches appears. You can select specific items to redact or discard others.
Safe Export:
  • The exported PDF is flattened (converted to images). Redacted data is permanently removed and cannot be uncovered by selecting text.

4. Auto-Filler

Fields & Markers:
  • Pick & Place: Select a data field (e.g., Name, Date) or a colored Marker from the sidebar. Click "Pick" and tap on the PDF to assign locations for that data.
  • Values: You can edit the text values in the sidebar, and they will update across all assigned locations.
Manual Tools:
  • Use the toolbar to stamp Text, Checkmarks, or Crossmarks anywhere on the document.
  • Eraser: Create white boxes to cover existing content.
Apply & Export:
  • Toggle "Apply Fields" or "Apply Markers" at the bottom to visualize the auto-filled data.
  • Click "Download PDF" to burn all fields and stamps into a new file.

5. File Compressor

Compression & Resizing:
  • Reduce Size: Shrink PDF documents and Images (JPG, PNG, WebP) to save space or meet upload requirements.
  • Image Options: Adjust quality percentage (10-90%) and resize dimensions (e.g., Full HD 1920px) for optimal balance between quality and file size.
Batch Processing:
  • Drag and drop multiple files to process them simultaneously.
  • Download files individually or grab them all at once as a ZIP archive.

한계점 및 Adobe와의 차이

이 앱은 업무를 보조하는 대체재(Substitute)이며, Adobe의 고도화된 기술력을 완전히 대체할 수는 없습니다. 다음과 같은 한계가 있습니다.

  • 비밀번호 문서: 비밀번호가 걸려있는 암호화된 문서는 Adobe의 보안 기술로 인해 열 수 없습니다.
  • 양식 (Fillable Forms): Adobe에서 생성된 특수 필기 가능 양식 등은 이 앱에서 인식하지 못할 수 있습니다.
  • "Aditor" (Add + Editor): 이름처럼 텍스트를 추가(Add)할 수는 있지만, PDF 원본의 글자를 수정(Edit)할 수는 없습니다.
    - 앱에서 새로 입력한 글자는 수정 가능합니다. - 원본 글자를 고치려면 '화이트(White-out)' 기능으로 지우거나, Auto-Filler의 불투명도(Opacity) 기능을 사용하여 텍스트를 덮어씌울 수 있습니다. - 텍스트 복사가 필요하다면 무료 Adobe Reader 사용을 권장합니다.
  • 엑셀 변환 (Export to Excel): 엑셀로 내보내기 기능은 없습니다. 대신 테이블을 잘라내기(Crop)하여 빈 페이지에 붙여넣은 후, 엑셀의 "Data from PDF" 기능이나 AI를 활용하면 변환이 가능합니다.

사용 설명서 (Tool Manuals)

1. File Preparer (파일 준비)

Merge (병합) 탭:
  • 여러 개의 PDF 또는 이미지 파일을 하나의 PDF로 합칩니다.
  • 정렬: 드래그 앤 드롭으로 순서를 바꾸거나, "Sort" 버튼을 눌러 중요 키워드(W2, 1099 등) 순으로 자동 정렬할 수 있습니다.
  • 옵션: 클릭 가능한 "목차(Table of Contents)"를 생성하고, 모든 페이지 하단에 페이지 번호를 매길 수 있습니다.
Extract (추출) 탭:
  • PDF: 문서에서 특정 페이지(예: "1-3, 5")만 뽑아냅니다. 결과물을 하나로 합치거나(Merge Result), 각각의 파일로 분리(Split Result)할 수 있습니다.
Split (분할) 탭:
  • 한 페이지를 수평 또는 수직으로 여러 조각으로 자릅니다 (예: 한 장에 두 개의 영수증이 있는 경우 분리).
  • 그리드에서 페이지를 선택한 후 "Apply Split Tool"을 누르면 미리보기 창에서 자르는 선을 조정할 수 있습니다.

2. PAGE ORGANIZER (정리 및 편집)

Organizer (페이지 관리):
  • 순서 변경: 페이지를 드래그 앤 드롭하여 순서를 바꿉니다.
  • 관리: 툴바를 사용해 페이지 회전, 삭제, 복제, 또는 빈 페이지 추가를 할 수 있습니다.
  • 가져오기: 여러 PDF와 이미지를 불러와 작업할 수 있습니다.
Write Note (메모 작성):
  • 원하는 위치에 메모(Note)를 남길 수 있습니다.

3. Redact OCR (개인정보 보호)

자동 감지 (Auto-Detection):
  • 키워드: "대외비" 같은 단어나 특정 번호를 입력하면 자동으로 찾아 마스킹합니다.
  • 주민/사회보장번호 (SSN): SSN 자동 감지를 켜면 번호 패턴을 찾아냅니다. 전체 마스킹 또는 부분 마스킹(XXX-XX-****)을 선택할 수 있습니다.
수동 도구 (Manual Tools):
  • 드래그 마스킹: 문서의 아무 곳이나 클릭하고 드래그하면 즉시 검은색으로 가려집니다.
  • 검토 (Review): 스캔 후 감지된 목록이 나타납니다. 원하는 항목만 선택하여 적용할 수 있습니다.
안전한 저장 (Safe Export):
  • 저장된 PDF는 이미지화(Flatten) 처리됩니다. 가려진 정보는 영구적으로 삭제되어, 텍스트 선택이나 복구로 알아낼 수 없습니다.

4. Auto-Filler (자동 입력)

필드 및 마커 (Fields & Markers):
  • 위치 지정 (Pick): 사이드바에서 데이터 필드(이름, 날짜 등)나 색상 마커를 선택합니다. "Pick" 버튼을 누른 뒤 PDF 위를 클릭하여 해당 데이터가 표시될 위치를 지정합니다.
  • 값 수정: 사이드바에서 텍스트 값을 수정하면 지정된 모든 위치에 일괄 적용됩니다.
수동 도구 (Manual Tools):
  • 상단 툴바를 이용해 텍스트, 체크(✓), X표시를 자유롭게 찍을 수 있습니다.
  • 지우개 (Eraser): 흰색 박스를 생성하여 기존 내용을 가릴 수 있습니다.
적용 및 저장:
  • 하단의 "Apply Fields" 또는 "Apply Markers" 버튼을 눌러 자동 입력된 데이터를 활성화합니다.
  • "Download PDF"를 누르면 모든 내용이 병합된 새 파일이 저장됩니다.

5. File Compressor (파일 압축)

압축 및 리사이징 (Compression):
  • 용량 축소: PDF 문서 및 이미지(JPG, PNG, WebP)의 파일 크기를 줄여 저장 공간을 아끼거나 업로드 제한을 준수할 수 있습니다.
  • 이미지 옵션: 화질(10-90%)을 조정하거나 해상도(예: Full HD 1920px)를 줄여 최적화할 수 있습니다.
일괄 처리 (Batch Processing):
  • 여러 파일을 한 번에 드래그하여 동시에 처리할 수 있습니다.
  • 처리된 파일을 개별적으로 다운로드하거나 ZIP 파일로 한 번에 받을 수 있습니다.
